    谓词（Predicate）¶

    [!note] 谓词就是"返回 bool 的可调用对象"——一个对输入做是非判断的东西。它从条件变量的等待条件、到 STL 算法的筛选规则、再到 C++20 Concepts 的约束表达式，贯穿了 C++ 的各个角落。

    1. 定义¶

    在 C++ 标准库语境中，谓词是一个可调用对象（callable），接受参数并返回 bool：

    bool isEven(int x) { return x % 2 == 0; }              // 函数
auto isOdd = [](int x) { return x % 2 != 0; };          // lambda
struct IsPos { bool operator()(int x) const { return x > 0; } };  // 函数对象

    2. 谓词在标准库中的三类核心应用¶

    2.1 条件变量：cv.wait(lock, pred)¶

    pred 就是传递给 wait() 的谓词。标准库在内部展开为：

    // cv.wait(lock, [] { return ready; });
// 等价于：
while (!ready) {
    cv.wait(lock);
}

    pred() 被调用的三种时机：

    时机 pred() 返回值 行为
    wait(pred) 入口 true 不挂起，直接返回（防御丢失唤醒）
    收到 notify 后 true 返回，继续执行
    虚假唤醒后 false 再次释放锁并挂起

    为什么必须是谓词？ 因为线程可能在没有满足条件的情况下醒来——虚假唤醒是操作系统层面的行为。谓词提供了"醒来后重新检查"的标准机制。

    2.2 STL 算法中的谓词¶

    std::vector<int> v = {1, 2, 3, 4, 5};

// 一元谓词（1 个参数）
auto it = std::find_if(v.begin(), v.end(), [](int x) { return x > 3; });

// 二元谓词（2 个参数）
std::sort(v.begin(), v.end(), [](int a, int b) { return a > b; });
    算法 谓词类型 作用
    find_if / count_if / remove_if 一元 筛选
    all_of / any_of / none_of 一元 全称/存在判断
    sort / stable_sort 二元 排序规则
    unique 二元 判断相邻元素是否"相同"

    2.3 C++20 Concepts 中的编译期谓词¶

    template<typename T>
concept Integral = std::is_integral_v<T>;  // 编译期谓词——判断类型是否满足约束

template<Integral T>
T twice(T v) { return v * 2; }  // T 必须满足 Integral 这个"谓词"

    3. 谓词的四种实现方式¶

    // ① 普通函数
bool isPositive(int x) { return x > 0; }

// ② 函数对象（Functor）——可携带状态
struct GreaterThan { int t; bool operator()(int x) const { return x > t; } };

// ③ Lambda —— C++11 起最常用
auto gt = [t](int x) { return x > t; };

// ④ std::function —— 类型擦除
std::function<bool(int)> f = gt;

    4. 工程约束：谓词必须是纯函数¶

    STL 算法假定谓词对相同输入永远返回相同结果——不能有副作用：

    // ❌ 带副作用的"谓词"——每次调用结果不同 → 未定义行为
int counter = 0;
auto bad = [&](int) { return counter++ < 5; };
std::remove_if(v.begin(), v.end(), bad);  // UB!
